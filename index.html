JavaScript is a client-side scripting language or browser scripting. Client-side scripting language means that the browser will run these scripts. Unlike client side, server side, code of server side languages ​​is run through web server. JavaScript is basically made up of 3 things. ECMAScript This is the core functionality of JavaScript. DOM It works with web page content 3. BOM It works with browsers.

The size of local storage is 10 MB, many say 5 MB. And the size of session storage is 5 MB. Local storage data has to be deleted manually. And if you close the session storage data browser or tab, the storage is cleared. Local storage works for any tab without working for any specific tab. And session storage works for a specific tab and when the tab is closed the data is deleted.
